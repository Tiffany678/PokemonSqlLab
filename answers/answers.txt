4. How many types are there?

5. How many pokemon have a secondary type?
    Select count(*) from pokemons where secondary_type >0;

6. What is each pokemon's primary type?
    Select p.name, t.name As primaryType from pokemons p Join types t ON p.id = t.id;

7. What is Rufflet's secondary type?
    Select name, secondary_type from pokemons where name = 'Rufflet';
